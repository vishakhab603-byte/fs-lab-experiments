# Experiment 1: Student Registration Form using HTML5 Validation

## Aim
To create a student registration form with HTML5 validation for name, email, and age inputs.

## Objective
- Design a responsive form using semantic HTML5
- Implement client-side validation using HTML5 attributes
- Style form elements with CSS3
- Handle validation feedback visually

## Tools Used
- HTML5
- CSS3
- VS Code
- Chrome Browser

## Description
This experiment demonstrates built-in HTML5 form validation techniques using attributes such as required, pattern, type=email, and min/max without using JavaScript.

## Output
The form validates user input and provides visual feedback for valid and invalid fields.
